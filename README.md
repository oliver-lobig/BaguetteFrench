# BaguetteFrench



https://github.com/user-attachments/assets/faf49c0c-3041-4523-be39-2ab89dd25c98


_BaguetteFrench_ is an open-source learning app made with Godot Engine.
It provides a variety of different ways to learn. There are functions build in to
motivate you.
The App is free and will always stay free of charge. It is a hobby project, that is waiting
for you help to take it to the next level.

Have fun and enjoy open-source!

## Distribution

The app can be found and installed via itch.io:
https://olli-games.itch.io/baguettefrench
